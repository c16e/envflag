Environment Definition Format

(dev|stage|prod) [URL]

Comments are also available. Comment line must start with #.

Example

prod www.example.com
stage stag.example.com
dev dev.example.com

# comments...
prod www.foobar.com
stage www.foobar.com/stag
dev www.foobar.com/dev
